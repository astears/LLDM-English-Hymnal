ENGLISH TRANSLATION



Los valientes soldados de Cristo,

Luchan contra el dragón infernal;

// Con el nombre de Cristo y su espada,

Esta lucha vamos a ganar //




CORO

// A la lucha // ya la guerra va a empezar,

Todos van a la vanguardia del cuerpo ministerial;

// Con el nombre de Cristo y su espada,

Esta lucha vamos a ganar //




2

Si tú quieres también ser soldado,

En Efesios te puedes armar,

En seis quince, está el banco de armas,

Allí encuentras a mi General;

Con el nombre de Cristo y su espada,

Esta lucha vamos a ganar.




3

Los valientes soldados de Cristo,

Esos nunca voltean para atrás;

Si la vida se pierde en la lucha,

Ya con Cristo la van a encontrar;

Con el nombre de Cristo y su espada,

Esta lucha vamos a ganar.
